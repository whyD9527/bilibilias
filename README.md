# BILIBILIAS

## 项目停止维护与仓库清理说明

BILIBILIAS 项目现已停止继续维护、公开提供与分发。

本项目于 2026 年 5 月 30 日起全面停止一切形式的公开发布 APK，学习移动端开发的开发者不得进行任何形式的分发产物。

具体原因请见：[停止 APK 发布声明](https://api.misakamoe.com/app/)

结合该情况，我们重新审视了 BILIBILIAS 过去及当前的项目形态。虽然本项目此前已停止公开发布 APK，主要以 GitHub 公开仓库代码形式存在，但相关代码、补丁、脚本及技术方案仍可能被他人自行构建、传播或使用，因此项目本身依旧存在较高的合规风险。

基于以上考虑，我们决定清理本仓库公开内容，仅保留本说明、历史 README 归档以及真实贡献者致谢信息。自本说明发布后，BILIBILIAS 不会再通过公开仓库、群聊、私发、网盘、镜像或其他渠道继续分发、维护或支持相关版本、代码、补丁、脚本及技术方案。

任何第三方流传的安装包、源码、镜像或修改版本均不代表 BILIBILIAS 官方立场，我们也不会再对其提供背书、维护或支持。请大家注意甄别来源，避免使用不可信版本。

感谢所有开发者、设计师和用户一直以来的参与与支持。BILIBILIAS 的初衷是帮助有需要的用户改善使用体验，但我们也尊重平台方的规则、权益和处理意见。后续我们会更加谨慎地评估项目边界，避免继续提供可能引发合规风险的功能、代码或技术方案。

## 立场

分享热爱，传递价值！

本项目自2021年发布，创建初期就将其不单单视作视频缓存工具，更重要的是它承载着分享文化和传递价值观的使命，正因如此它才与众不同。
我想说的是，其他Fork、同名仓库、同名APP项目均与本项目无关，因为立场存在不同，目标并不相同，BILIBILIAS拥有自己的价值观，我们不应该将其简单的将他们画上等号。
